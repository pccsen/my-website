## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization

### Update Contact Information

Edit `components/Contact.tsx` and replace the placeholder contact information:
- Email
- LinkedIn profile URL
- Telegram username
- GitHub username

### Update Projects

Edit `components/Projects.tsx` to add or modify your projects.

### Update Skills

Edit `components/Skills.tsx` to customize your skills list.

## Build for Production

```bash
npm run build
npm start
```
